# Soul Memory

Access the persistent chitta soul memory system. Recall past conversations, store learnings, and get contextually relevant memories for your current task.

## Usage

Recall memories relevant to a topic:
```
Use mcp__chitta_bridge__soul_recall with query="MAG recovery from ancient permafrost"
```

Remember something for later:
```
Use mcp__chitta_bridge__soul_remember with content="Gemma4:26b works well for bioinformatics reasoning in rooms"
```

Get smart context for a task (auto-retrieves relevant memories):
```
Use mcp__chitta_bridge__soul_context with task="optimize the assembly pipeline"
```

Check soul status:
```
Use mcp__chitta_bridge__soul_status
```
