opencode-bridge

linking Claude Code to OpenCode & Codex

multi-model consultation · agentic execution · domain framing

pip install opencode-bridge
Claude Code → MCP → OpenCode Hub → Model → Hub → Claude Code
01

Install

claude-code 
$ pip install opencode-bridge

requires: Python 3.10+ · Claude Code · OpenCode CLI and/or Codex CLI

then add to your MCP config:

{ "opencode-bridge": { "command": "opencode-bridge" } }

How It Works

1

2
opencode-bridge auto-detects domain
3
routes to OpenCode with expert framing
4
OpenCode consults model
5
response synthesized and returned to Claude

Multi-Agent Orchestration

Coordinate workflows across multiple AI backends with hierarchical agent patterns.

multi_consult fan-out question to OpenCode + Codex in parallel · synthesize results
agent_chain sequential steps · {previous} passes result forward · plan → implement → review
delegate_codex delegate to Codex · optional OpenCode review · Claude → Codex → OpenCode
parallel_agents run multiple tasks concurrently · across both backends
1
Claude Code
2
OpenCode (plan)
3
Codex (implement)
4
OpenCode (review)
agent_chain example 
agent_chain(steps=[
  {"backend": "opencode", "task": "Plan caching strategy", "agent": "plan"},
  {"backend": "codex", "task": "Implement: {previous}"},
  {"backend": "opencode", "task": "Review: {previous}"}
])
08

Discussion Rooms

An async multi-agent roundtable — any mix of OpenCode and Codex sessions, any models. All participants respond in parallel each round, each seeing the full thread before posting.

room_create create a room with named participants · reuse existing sessions or spin up new ones
room_add_participant add a new agent to an ongoing discussion
room_run run N rounds · all agents respond concurrently · full thread visible to each
room_read read full transcript at any time
discussion room example 
room_create(
  room_id="arch-review",
  topic="Should we use event sourcing here?",
  participants='[
    {"name":"Codex","backend":"codex","session_id":"codex-1"},
    {"name":"Gemini","backend":"opencode","session_id":"gemini-1"},
    {"name":"GPT54","backend":"opencode","model":"openai/gpt-5.4"}
  ]'
)
room_run(room_id="arch-review")   # all 3 respond in parallel
room_read(room_id="arch-review")  # full transcript
room_create create a room with named participants
room_add_participant add agent to ongoing discussion
