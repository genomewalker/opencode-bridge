Install

claude-code 
$ pip install chitta-bridge

requires: Python 3.10+ · Claude Code · one or more backends (OpenCode, Codex, Ollama)

then add to your MCP config:

{ "chitta-bridge": { "command": "chitta-bridge" } }

Multi-Agent Orchestration

Discussion Rooms

An async multi-agent roundtable — any mix of backends and models. All participants respond in parallel each round, each seeing the full thread before posting. A synthesis pass distills consensus, disagreements, and the best answer.

room_create create a room with named participants · reuse existing sessions or spin up new ones
room_add_participant add a new agent to an ongoing discussion
room_run run N rounds · all agents respond concurrently · full thread visible to each
room_synthesize distill the transcript · consensus · disagreements · best answer · open questions
room_read read full transcript at any time
discussion room example 
room_create(
  room_id="arch-review",
  topic="Should we use event sourcing here?",
  participants='[
    {"name":"Codex","backend":"codex","session_id":"codex-1"},
    {"name":"Gemini","backend":"opencode","session_id":"gemini-1"},
    {"name":"Llama","backend":"local","model":"qwen3:30b-a3b","base_url":"http://gpunode:11434/v1"}
  ]'
)
room_run(room_id="arch-review")        # all 3 respond in parallel
room_synthesize(room_id="arch-review") # distill into one answer
room_create create a room with named participants
room_add_participant add agent to ongoing discussion
room_run run N parallel rounds
room_synthesize distill transcript into one answer
room_read read full transcript

Local GPU Models

Chat with Ollama or vLLM running on GPU nodes — no cloud API needed. Auto-discovers endpoints via Slurm job queue, cached URL files, or environment variable.

cached URLs /tmp/ollama-server-*.url files written by slurm-serve-ollama.sh
Slurm jobs squeue --me scans your running GPU jobs for Ollama endpoints
env var CHITTA_BRIDGE_GPU_NODES=node1,node2
fallback localhost:11434
local model session 
local_discover()  # find GPU nodes and loaded models

local_start(session_id="gpu1", model="qwen3:30b-a3b")
local_discuss(message="Explain cache invalidation strategies")

# explicit endpoint
local_start(session_id="gpu2", model="qwen2.5:32b",
            endpoint="http://gpunode01:11434/v1")
local_discover find GPU nodes with Ollama/vLLM running
local_start start session (auto-discovers endpoint)
